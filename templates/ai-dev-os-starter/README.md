# AI Dev OS Starter Template

A minimal configuration template for AI Dev OS.

## Structure

```text
ai-dev-os/
└── 03_guidelines/
    └── common/
        └── code.md    # Basic coding rules
```

## Usage

1. Copy this template into your project
2. Edit `code.md` to match your project
3. Add a reference from AGENTS.md
4. Add layers as needed (01_philosophy/, 02_decision-criteria/, etc.)

## Next Steps

- Invoke `#ai-dev-os-check` in Kiro chat to verify guideline compliance
- Invoke `#ai-dev-os-extract` to extract rules from code reviews
- When rules grow, migrate to the `ai-dev-os-full` template
